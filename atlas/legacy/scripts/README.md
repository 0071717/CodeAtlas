# Legacy Scripts

These scripts are retained from the older prompt-first CodeAtlas workflow.

They are intentionally separated from active runners so Kiro does not mistake them for the current deterministic V2 path.

Use active runners from `atlas/scripts/` first:

```bash
bash atlas/scripts/run-pre-transfer-check.sh
bash atlas/scripts/run-framework-v2-suite.sh
```

Use these legacy scripts only when explicitly requested.
