# CodeAtlas Legacy Area

This folder preserves older prompt-first and map-first orchestration material.

These files are **not** the canonical CodeAtlas V2 execution path.

Canonical first-run path:

```bash
python3 atlas/tools/codeatlas_v2_canonical.py doctor
python3 atlas/tools/codeatlas_v2_canonical.py all
```

Legacy scripts can be useful for historical reference, exploratory Kiro work, or future migration, but they must not be treated as authoritative extraction for:

```text
endpoint inventory
route inventory
OpenSearch Query DSL
permissions
runtime envelopes
error flows
coverage links
business rules
requirements
```
